---
description: Deep-analyze a single paper into a wiki entry
---

Invoke the paper-wiki analyze skill.

Accept the paper as an arXiv id (`arxiv:1234.5678`), an arXiv URL, a
Semantic Scholar id (`s2:<paperId>`), or a fuzzy title. Confirm the
resolution before generating the note. If the user invoked the command
with no argument, ask which paper they want to analyze (offer recent
digest entries as candidates if available).

After writing the note, confirm the vault path it landed in.
